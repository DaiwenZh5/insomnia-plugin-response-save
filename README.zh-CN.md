## 安装
从 "Preferences > Plugins" 安装 `insomnia-plugin-response-export` 插件。

## 使用
点击一个 "request" 选中 "保存"，将会触发该请求，并保存响应流到文件。
> 与 "Preview" 按钮不同的是，该文件将会使用 "Content-Disposition" 描述值命名。
